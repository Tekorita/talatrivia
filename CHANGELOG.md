# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-16

### Added
- Initial project structure
- FastAPI backend with hexagonal architecture
- Docker and docker-compose setup for local development
- PostgreSQL integration
- Alembic migrations
- Domain entities (User, Trivia, Question, Option, TriviaQuestion, Participation, Answer)
- SQLAlchemy models with proper constraints and relationships
- Health check endpoint
- VSCode/Cursor debug configuration
- Monorepo structure with docker-compose at root level
